# Convertisseur Montant en Lettres

Un petit logiciel en **Lazarus (Free Pascal)** qui convertit les chiffres en lettres en français, avec support de plusieurs devises.

## Fonctionnalités

- Conversion chiffre → lettres (français correct)
- Formatage automatique du montant (1 234,56)
- Gestion du singulier/pluriel (euro/euros, centime/centimes...)
- Support de plusieurs devises (Euro, Dollar, Dinar, etc.)
- Touche **Entrée** pour convertir
- Touche **Échap** pour nouveau
- Copie dans le presse-papiers
- Interface simple et propre

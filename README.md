# Chitre v3.0 - Convertisseur Montant en Lettres

![Lazarus](https://img.shields.io/badge/Lazarus-2.2.0+-blue?style=flat-square&logo=pascal)
![Windows](https://img.shields.io/badge/Windows-10%20%7C%2011-success?style=flat-square)
![License](https://img.shields.io/badge/License-MIT-yellow.svg)

Un convertisseur **chiffres en lettres** en français, développé avec **Lazarus (Free Pascal)**.  
Idéal pour les factures, les chèques, les quittances et les documents administratifs.

---

## ✨ Fonctionnalités

- Conversion précise des nombres en lettres françaises
- Gestion automatique du **singulier/pluriel** (euro/euros, centime/centimes...)
- Formatage en temps réel du montant (`1 234,56`)
- Support de plusieurs devises (Euro, Dollar, Dinar, Livre, Dirham, Riyal...)
- Interface simple et ergonomique
- Raccourcis clavier :
  - **Entrée** → Convertir
  - **Échap** → Nouveau
- Copie rapide dans le presse-papiers
- Mise en majuscule de chaque mot

## 🛠 Comment compiler depuis les sources

1. Ouvrir **Lazarus IDE**
2. Charger le projet : `Chitre_V3.lpi`
3. Appuyer sur **`F9`** pour compiler
4. L'exécutable sera généré dans le dossier du projet
